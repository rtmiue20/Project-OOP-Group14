# Project-OOP-Group14
TOPIC: Xây dựng Chương trình quản lý công tác Đoàn/Hội

NNLT: C#
  Minh họa 4 tính chất OOP trong bài
Encapsulation (Đóng gói): Tất cả biến (field) để private, chỉ sửa qua Property.
Inheritance (Kế thừa): Student kế thừa Human; Offcial kế thừa Student.
Polymorphism (Đa hình): List danh sách chứa chung cả Student và Offcial. Khi gọi hàm TinhDiem(), chương trình tự biết đối tượng nào tính theo công thức nào.
Abstraction (Trừu tượng): Không thể tạo đối tượng new Human() mà phải tạo new Student().
